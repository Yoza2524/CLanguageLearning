## 环境配置(Windows)
系统环境使用 [Scoop](https://github.com/ScoopInstaller/Scoop) 下载
1. 解释器
```Powershell
scoop install mingw
```
2. 编辑器(VSCode)
```Powershell
scoop install vscode
```
- 编辑器插件: 
    + [C/C++](https://github.com/microsoft/vscode-cpptools)
    + [Code Runner](https://github.com/formulahendry/vscode-code-runner)
